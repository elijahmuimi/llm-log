## 📋 What is llm-log?

llm-log is a lightweight tool designed to help track and save data from calls you make to large language model (LLM) APIs. It creates clear, easy-to-read log files using a common format called JSONL. This makes it simple to review or share the history of your API interactions.

This tool works without the need for any extra software. It comes as a single file you can add into your projects. If you want to see when and how your LLM API was used, llm-log gives you a straightforward record.
